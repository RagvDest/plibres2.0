Introducción

Conocimiento previo

Un algoritmo es un conjunto ordenado de operaciones sistemáticas que permite hacer un cálculo y hallar la solución de un tipo de problemas.

Quick Sort es conocido algoritmo de ordenamiento, el cual tiene como objetivo es ordenar una lista.

Complejidad:

La complejidad de un algoritmo se mide utilizando el análisis asintótico, los resultados de este análisis se muestran a continuación de menor a mayor complejidad.

  • O(1)
  • O(n)
  • O(n log (n))
  • O(n^z)
  • O(n^n)

En la práctica, es el algoritmo de ordenación más rápido conocido, su tiempo de ejecución promedio es O(n log (n)), siendo en el peor de los casos O(n2), caso altamente improbable. El hecho de que sea más rápido que otros algoritmos de ordenación con tiempo promedio de O(n log (n)) ( como SmoothSort o HeapSort ) viene dado por que QuickSort realiza menos operaciones ya que el método utilizado es el de partición.
