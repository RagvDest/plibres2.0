Ejercicio Factorial Recursivo

Desarrollar el código en java que permita calcular el factorial de un número de manera recursiva.

Solución:

Para entender mejor hay que saber que la función recursiva de factorial es la siguiente (n) = n * factorial(n-1)
El código sería el siguiente

import java.util.Scanner;
 
public class FactorialRecursivo {
     
    public static void main(String[] args) {
     
        Scanner sc = new Scanner(System.in);
        System.out.print("Introduce un número: ");
        int num = sc.nextInt();
        sc.close();
         
        System.out.println("\nEl factorial de " + num + " es: " + factorial(num));
    }
     
    public static int factorial(int num){
        if(num == 0){
            return 1;
        }
        else
            return num * factorial(num-1);
    }
}
